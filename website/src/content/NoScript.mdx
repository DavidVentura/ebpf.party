The interactive code editor requires JavaScript to function.

You can use your own editor locally if you prefer.

The API is:

- POST /register
  - Body: `{"user-id": "a UUIDv7", "alias": "your alias"}`
- POST /exercise
  - Headers: `"X-User-Id": "your UUIDv7"`
- POST /submit_answer
  - Headers: `"X-User-Id": "your UUIDv7"`
  - Body: `{"answer": "your answer"}`
